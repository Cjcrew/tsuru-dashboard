How to create an application

Creating an application

Before deploy

After creating your application, you will receive a git url for deploy.

Info

App url: http://appname.cloud.tsuru.io

Git clone url: git@cloud.tsuru.io:appname.git

Platform: python

How to deploy

Add your application git url as a remote target in your git.

'git remote add tsuru git@cloud.tsuru.io:appname.git
